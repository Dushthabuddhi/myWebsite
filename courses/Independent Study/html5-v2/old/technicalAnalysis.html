Major Development Tasks

Editor

The editor will be where most of the time is spent on this game during development. The editor features 4 different editing features.

  • Terrain Editor
  • Animation, Modeling, Particle Effects Editor/Viewer
  • User Interface Resource Editor
  • Font Pack Editor and Builder

First Playable Requirements

The first playable game should include

  • Art development
  • 3D graphics engine optimization
  • User Interface Resource creation
  • Terrain Creation
  • AI implementation
  • Combat implementation
  • Physics implementation
